# IT Solutions Trade & Service — Website

Corporate portfolio site for IT Solutions Trade & Service: IT accessories,
CCTV, solar, and networking products and installation services. Links out to
the company's separate online store — this site itself does not sell.

## Quick start

```bash
npm install
cp .env.example .env.local
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

```bash
npm run dev         # local dev server
npm run build       # production build
npm run lint        # ESLint
npm run typecheck   # TypeScript check
npm run test        # Vitest
npm run format      # Prettier
```
